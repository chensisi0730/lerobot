# ACT (动作分块 Transformer)

ACT 是一种**轻量级高效的模仿学习策略**，特别适用于精细操作任务。由于其训练时间短、计算要求低且性能优异，**当你刚开始使用 LeRobot 时，我们推荐你首先使用 ACT**。

## 模型概述

动作分块 Transformer (Action Chunking with Transformers, ACT) 来自论文 [Learning Fine-Grained Bimanual Manipulation with Low-Cost Hardware](https://arxiv.org/abs/2304.13705)（Zhao 等人）。该策略旨在使用低成本硬件和少量演示数据实现精确、接触丰富的操作任务。

### 为什么 ACT 非常适合初学者

ACT 作为优秀的起点有以下几个原因：

- **训练快速**：在单个 GPU 上几个小时即可完成训练
- **轻量级**：仅约 8000 万参数，高效且易于使用
- **数据高效**：通常只需 50 个演示就能获得高成功率

### 架构

ACT 使用基于 Transformer 的架构，包含三个主要组件：

1. **视觉骨干网络**：ResNet-18 处理来自多个相机视角的图像
2. **Transformer 编码器**：综合来自相机特征、关节位置和学习的潜在变量的信息
3. **Transformer 解码器**：使用交叉注意力生成连贯的动作序列

策略的输入包括：

- 多个 RGB 图像（例如来自手腕相机、前/顶部相机）
- 当前机器人关节位置
- 潜在风格变量 `z`（训练期间学习，推理时设为零）

输出：`k` 个未来动作序列的分块。

## 安装要求

1. 按照我们的[安装指南](./installation)安装 LeRobot。
2. ACT 已包含在基础 LeRobot 安装中，无需额外依赖！

## 训练 ACT

ACT 与标准 LeRobot 训练流程无缝集成。以下是在你的数据集上训练 ACT 的完整示例：

```bash
lerobot-train \
  --dataset.repo_id=${HF_USER}/your_dataset \
  --policy.type=act \
  --output_dir=outputs/train/act_your_dataset \
  --job_name=act_your_dataset \
  --policy.device=cuda \
  --wandb.enable=true \
  --policy.repo_id=${HF_USER}/act_policy
```

### 训练技巧

1. **从默认参数开始**：ACT 的默认超参数对大多数任务效果良好
2. **训练时长**：在单个 GPU 上预期 100k 训练步需要几个小时
3. **批量大小**：从批量大小 8 开始，根据 GPU 内存进行调整

### 使用 Google Colab 训练

如果你的本地电脑没有强大的 GPU，你可以按照 [ACT 训练笔记本](./notebooks#training-act) 使用 Google Colab 训练你的模型。

## 评估 ACT

训练完成后，你可以使用 `lerobot-record` 命令配合训练好的策略来评估你的 ACT 策略。这将运行推理并录制评估 episode：

```bash
lerobot-record \
  --robot.type=so100_follower \
  --robot.port=/dev/ttyACM0 \
  --robot.id=my_robot \
  --robot.cameras="{ front: {type: opencv, index_or_path: 0, width: 640, height: 480, fps: 30}}" \
  --display_data=true \
  --dataset.repo_id=${HF_USER}/eval_act_your_dataset \
  --dataset.num_episodes=10 \
  --dataset.single_task="Your task description" \
  --dataset.streaming_encoding=true \
  --dataset.encoder_threads=2 \
  # --dataset.vcodec=auto \
  --policy.path=${HF_USER}/act_policy
```
