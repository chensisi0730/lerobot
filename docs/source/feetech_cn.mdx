# Feetech 电机固件更新

本教程将指导你使用官方 Feetech 软件更新 Feetech 电机的固件。

## 前置条件

- Windows 电脑（Feetech 软件仅支持 Windows）
- Feetech 电机控制板
- 用于将控制板连接到电脑的 USB 线
- 连接到控制板的 Feetech 电机

## 第一步：下载 Feetech 软件

1. 访问官方 Feetech 软件下载页面：[https://www.feetechrc.com/software.html](https://www.feetechrc.com/software.html)
2. 下载最新版本的 Feetech 调试软件 (FD)
3. 在 Windows 电脑上安装该软件

## 第二步：硬件设置

1. 将 Feetech 电机连接到电机控制板
2. 通过 USB 线将电机控制板连接到 Windows 电脑
3. 确保电机已供电

## 第三步：配置连接

1. 启动 Feetech 调试软件
2. 从端口下拉菜单中选择正确的 COM 端口
   - 如果不确定使用哪个端口，请在 Windows 设备管理器的"端口 (COM & LPT)"下查看
3. 设置适当的波特率（大多数 Feetech 电机通常为 1000000）
4. 点击"Open"建立与控制板的通信

## 第四步：扫描电机

1. 连接后，点击"Search"按钮检测所有已连接的电机
2. 软件将自动发现并列出总线上的所有电机
3. 每个电机将显示其 ID 编号

## 第五步：更新固件

对于每个要更新的电机：

1. **从列表中选择电机**（点击它）
2. **点击 Upgrade（升级）选项卡**：
3. **点击 Online（在线）按钮**：
   - 如果找到潜在固件更新，它将显示在框中
4. **点击 Upgrade（升级）按钮**：
   - 将显示更新进度

## 第六步：验证更新

1. 更新完成后，软件应自动刷新电机信息
2. 验证固件版本已更新到预期版本

## 重要注意事项

⚠️ **警告**：在固件更新期间请勿断开电源或 USB 连接，否则可能会损坏电机。

## 附加：Linux/macOS 上的电机调试

仅用于调试目的，你可以使用开源 Feetech 调试工具：

- **仓库**：[FT_SCServo_Debug_Qt](https://github.com/CarolinePascal/FT_SCServo_Debug_Qt/tree/fix/port-search-timer)

### 安装说明

按照仓库中的说明安装该工具，对于 Ubuntu，你可以直接安装；对于 macOS，你需要从源码构建。

**限制：**

- 此工具仅用于调试和参数调整
- 固件更新仍必须在 Windows 上使用官方 Feetech 软件进行
