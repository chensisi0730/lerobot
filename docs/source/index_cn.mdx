## 概述

🤗 LeRobot 旨在为 PyTorch 中的真实机器人提供模型、数据集和工具。我们的目标是降低机器人学的门槛，让每个人都能从共享数据集和预训练模型中受益。

🤗 LeRobot 包含了已证明可以迁移到真实世界的 state-of-the-art 方法，重点关注模仿学习和强化学习。

🤗 LeRobot 提供了一系列预训练模型、包含人类演示的数据集和仿真环境，让每个人都能快速上手。

🤗 LeRobot 在 HuggingFace 页面上托管预训练模型和数据集。

欢迎加入 [Discord](https://discord.gg/s3KuuzsPFb) 上的 LeRobot 社区！

## 快速开始

### 安装

```bash
# 安装 LeRobot
pip install lerobot

# 验证安装
lerobot-info
```

### 数据集

```python
from lerobot.datasets import LeRobotDataset

# 从 Hub 加载数据集
dataset = LeRobotDataset("lerobot/aloha_mobile_cabinet")

# 访问数据
episode_index = 0
print(f"{dataset[episode_index]['action'].shape=}\n")
```

### 训练

```bash
# 训练策略
lerobot-train \
    --policy=act \
    --dataset.repo_id=lerobot/pusht
```

### 推理

```python
from lerobot.policies import ACTPolicy

# 加载预训练模型
policy = ACTPolicy.from_pretrained("lerobot/act_pusht")

# 推理
action = policy.select_action(observation)
```

## 支持的策略

| 策略 | 类型 | 特点 |
|------|------|------|
| **ACT** | 模仿学习 | 动作分块 + Transformer |

## 数据集格式

LeRobot 使用标准化的数据集格式：

```
dataset/
├── data/                    # Parquet 文件
│   └── chunk-XXX/
│       └── file-XXX.parquet
├── meta/                    # 元数据
│   ├── info.json           # 数据集信息
│   ├── stats.json          # 统计信息 (归一化)
│   └── tasks.parquet       # 任务描述
└── videos/                  # MP4 视频
    └── observation.images.XXX/
```
