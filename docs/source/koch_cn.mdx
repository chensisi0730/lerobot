# Koch v1.1

下面的步骤将解释如何组装 Koch v1.1 机器人。

## 订购和组装零件

请遵循此 [README](https://github.com/jess-moss/koch-v1-1) 中提供的采购和组装说明。这将引导你完成从臂和主臂的设置，如下图所示。

有关组装过程的视觉教程，你可以参考[此视频教程](https://youtu.be/8nQIg9BwwTk)。

> [!警告]
> 自该视频制作以来，我们简化了配置阶段。因此，有两件事与视频中的说明不同：

> - 不要立即插入所有电机电缆，请在[配置电机](#configure-the-motors)中按照说明进行操作。
> - 请勿立即将控制板（PCB）螺丝固定到底座，请在[配置电机](#configure-the-motors)中按照说明进行操作。

## 安装 LeRobot 🤗

要安装 LeRobot，请遵循我们的[安装指南](./installation)

除这些说明外，你还需要安装 Dynamixel SDK：

```bash
pip install -e ".[dynamixel]"
```

## 配置电机

### 1. 找到每个臂对应的 USB 端口

要找到每个总线伺服适配器的端口，请运行此脚本：

```bash
lerobot-find-port
```

<hfoptions id="example">
<hfoption id="Mac">

示例输出：

```
Finding all available ports for the MotorBus.
['/dev/tty.usbmodem575E0032081', '/dev/tty.usbmodem575E0031751']
Remove the USB cable from your MotorsBus and press Enter when done.

[...Disconnect corresponding leader or follower arm and press Enter...]

The port of this MotorsBus is /dev/tty.usbmodem575E0032081
Reconnect the USB cable.
```

找到的端口是：`/dev/tty.usbmodem575E0032081`，对应你的主臂或从臂。

</hfoption>
<hfoption id="Linux">

在 Linux 上，你可能需要通过运行以下命令来授予 USB 端口访问权限：

```bash
sudo chmod 666 /dev/ttyACM0
sudo chmod 666 /dev/ttyACM1
```

示例输出：

```
Finding all available ports for the MotorBus.
['/dev/ttyACM0', '/dev/ttyACM1']
Remove the usb cable from your MotorsBus and press Enter when done.

[...Disconnect corresponding leader or follower arm and press Enter...]

The port of this MotorsBus is /dev/ttyACM1
Reconnect the USB cable.
```

找到的端口是：`/dev/ttyACM1`，对应你的主臂或从臂。

### 2. 设置电机 ID 和波特率

每个电机在总线上都有一个唯一的 ID。新电机通常默认 ID 为 `1`。为了让电机与控制器之间的通信正常工作，我们首先需要为每个电机设置一个唯一的、不同 ID。此外，总线上数据传输的速度由波特率决定。为了让控制器和所有电机能够相互通信，它们需要配置相同的波特率。

为此，我们需要首先单独连接每个电机来设置这些参数。由于我们将把这些参数写入电机内部存储器的非易失性部分（EEPROM），我们只需要做一次。

如果你是从其他机器人上重复使用电机，你可能也需要执行此步骤，因为 ID 和波特率可能不匹配。

#### 从臂

将电脑的 USB 电缆和 5V 电源连接到从臂的控制板。然后，运行以下命令或使用上一步获得的端口运行 API 示例。你还需要使用 `id` 参数为你的主臂命名。

有关如何设置电机 ID 的视觉参考，请参阅[此视频](https://huggingface.co/docs/lerobot/en/so101#setup-motors-video)，其中我们展示了 SO101 臂的设置过程。

<hfoptions id="setup_motors">
<hfoption id="命令">

```bash
lerobot-setup-motors \
    --robot.type=koch_follower \
    --robot.port=/dev/tty.usbmodem575E0031751 # <- 在这里粘贴上一步找到的端口
```

</hforiginal>
<hfoption id="API 示例">

```python
from lerobot.robots.koch_follower import KochFollower, KochFollowerConfig

config = KochFollowerConfig(
    port="/dev/tty.usbmodem575E0031751",
    id="my_awesome_follower_arm",
)
follower = KochFollower(config)
follower.setup_motors()
```

</hforiginal>
</hforiginals>

你应该看到以下说明。

```
Connect the controller board to the 'gripper' motor only and press enter.
```

按照说明，插入夹爪的电机。确保它是唯一连接到控制板的电机，并且该电机本身尚未串联到任何其他电机。当你按 `[Enter]` 时，脚本将自动为该电机设置 ID 和波特率。

<details>
<summary>故障排除</summary>

如果此时出错，请检查你的电缆并确保它们已正确插入：

<ul>
  <li>电源</li>
  <li>电脑和控制板之间的 USB 电缆</li>
  <li>控制板到电机的 3 针电缆</li>
</ul>

如果你使用的是 Waveshare 控制板，请确保两个跳线设置在 `B` 通道（USB）上。

</details>

然后你应该看到以下消息：

```
'gripper' motor id set to 6
```

接下来是下一条说明：

```
Connect the controller board to the 'wrist_roll' motor only and press enter.
```

你可以从控制板断开 3 针电缆的连接，但可以将其保持在夹爪电机的另一端，因为它已经在正确的位置。现在，将另一根 3 针电缆插入腕部旋转电机并连接到控制板。与之前的电机一样，确保它是唯一连接到控制板的电机，并且该电机本身没有连接到任何其他电机。

按照说明为每个电机重复此操作。

> [!提示]
> 在按 Enter 之前，检查每个步骤的布线。例如，电源电缆可能在操作控制板时断开连接。

完成后，脚本将简单结束，此时电机已准备好使用。你现在可以将每个电机的 3 针电缆连接到下一个电机，并将第一个电机（ID=1 的 'shoulder pan'）的电缆连接到控制板，控制板现在可以连接到臂的底座。

#### 主臂

对主臂执行相同的步骤，但相应地修改命令或脚本。

<hfoptions id="setup_motors">
<hfoption id="命令">

```bash
lerobot-setup-motors \
    --teleop.type=koch_leader \
    --teleop.port=/dev/tty.usbmodem575E0031751 \  # <- 在这里粘贴上一步找到的端口
```

</hforiginal>
<hfoption id="API 示例">

```python
from lerobot.teleoperators.koch_leader import KochLeader, KochLeaderConfig

config = KochLeaderConfig(
    port="/dev/tty.usbmodem575E0031751",
    id="my_awesome_leader_arm",
)
leader = KochLeader(config)
leader.setup_motors()
```

## 校准

接下来，你需要校准你的机器人，以确保主臂和从臂在相同物理位置时具有相同的位姿值。
校准过程非常重要，因为它允许在一个机器人上训练的神经网络在另一个机器人上工作。

#### 从臂

运行以下命令或 API 示例来校准从臂：

<hfoptions id="calibrate_follower">
<hfoption id="命令">

```bash
lerobot-calibrate \
    --robot.type=koch_follower \
    --robot.port=/dev/tty.usbmodem58760431551 \ # <- 你的机器人端口
    --robot.id=my_awesome_follower_arm  # <- 给机器人一个唯一名称
```

</hforiginal>
<hfoption id="API 示例">

```python
from lerobot.robots.koch_follower import KochFollowerConfig, KochFollower

config = KochFollowerConfig(
    port="/dev/tty.usbmodem585A0076891",
    id="my_awesome_follower_arm",
)

follower = KochFollower(config)
follower.connect(calibrate=False)
follower.calibrate()
follower.disconnect()
```

</hforiginal>
</hforiginals>

我们统一了大多数机器人的校准方法。因此，这个 Koch 臂的校准步骤与 SO100 和 SO101 的步骤相同。首先，我们必须将机器人移动到每个关节都在其范围中间的位置，然后按 `Enter`。其次，我们将所有关节移动到其整个运动范围。SO101 的相同过程视频可在此处找到[参考](https://huggingface.co/docs/lerobot/en/so101#calibration-video)。

#### 主臂

对主臂执行相同的校准步骤，运行以下命令或 API 示例：

<hfoptions id="calibrate_leader">
<hfoption id="命令">

```bash
lerobot-calibrate \
    --teleop.type=koch_leader \
    --teleop.port=/dev/tty.usbmodem58760431551 \ # <- 你的机器人端口
    --teleop.id=my_awesome_leader_arm  # <- 给遥操作设备一个唯一名称
```

</hforiginal>
<hfoption id="API 示例">

```python
from lerobot.teleoperators.koch_leader import KochLeaderConfig, KochLeader

config = KochLeaderConfig(
    port="/dev/tty.usbmodem575E0031751",
    id="my_awesome_leader_arm",
)

leader = KochLeader(config)
leader.connect(calibrate=False)
leader.calibrate()
leader.disconnect()
```
